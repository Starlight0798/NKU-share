课程简介:	本课程从实用的角度出发，通过介绍工业界广泛使用的网络安全实例，概览了当前网络安全解决方案中使用的主要算法，重要协议和系统管理方法。主要内容包括：(1) (略讲)密码算法和协议，包括网络安全应用中最常用的密码算法和协议; (2)网络安全应用，介绍了网络安全解决方案中使用的各种安全协议，如Kerberos, PGP, S/MIME, IPSec, SSL/TLS，以及SET等; (3) (选讲)系统安全，介绍了一些系统级的安全问题，如网络入侵，恶意软件和防火墙等。 课程的教学对象是信息安全专业、计算机专业和物联网专业三年级本科生。通过学习本课程，培养学生综合应用以前所学的信息安全基础知识，理解和构建安全的网络系统的能力。

英文简介:	It is the purpose of this course to provide a practical survey of network security applications and standards. By giving a great deal of examples and instances of network security solutions, we mainly focus on applications that are widely deployed on Internet and introduce prevalent industrial standards accordingly. This course includes three main parts: (1) Cryptography algorithms and protocols, including encryption, hash functions, digital signatures algorithms and key exchange protocol; (2) Network security applications, covering important network security tools and applications, such as Kerberos, X.509 v3 certificates, PGP, S/MIME, IP Security, SSL/TLS, SET; (3) (Optional) System security, focusing on system-level security issues, including threats and countermeasures for intruders and viruses, and the use of firewalls and trusted systems. This course is suitable for the sophomores who major in Information Security, Computer Science and The Internet of Things. When the students finished the course, they should have a basic idea with cryptography, system and network security, and should have the ability to build a secure network system.

教学目标:	本课程的教学目的是进一步加强学生对网络安全技术的认识，并通过理论学习和上机实验使学生具备构建安全的网络系统的能力。 通过本课程的学习，学生应能掌握在网络安全应用中最常用的密码算法和协议；掌握各种网络安全应用以及网络安全解决方案中使用的各种安全协议，如Kerberos, PGP, S/MIME, IPSec,SSL/TLS，以及SET等；能够解决一些系统级的安全问题，如网络入侵，恶意软件和防火墙配置等。

考核方式:	本课程的期末考核方式是提交一份综合性研究报告，报告需要在本课程涵盖的某一个方向上有比较深入的调研。研究报告的成绩和平时成绩（包括平时上机和书面作业成绩）比例为5:5。
